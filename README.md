# Gulf Fish Phylogenetics

A static educational website exploring the evolutionary relationships of Gulf of Mexico fish species using COI DNA barcodes. Built as a submission for **MARB 668 601: Evolutionary Biology**.

Code to replicate the phylogenetic tree building (NJ, ML, and Bayesian inference) is found in the `R/` folder.

## AI Statement

Generative AI tools were used to help construct the website architecture. The ideas presented are my own original thoughts and work.
